Compressor Mod
==================
Compressor turns carbon into diamonds with some time!

Crafting
--------
	SI	X	 SI
	SI	X	 SI
	SI	MC	 SI

SI : Steel Ingot

MC : Mese Crystal

X  : Empty/Air
